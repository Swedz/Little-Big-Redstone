# Введение в логику

<FloatingColumn align="right">
	### Оглавление
	<LogicIndex />
</FloatingColumn>

Логические компоненты, или логические вентели, — это то, что управляет вашей схемой. Все логические  
компоненты имеют порты ввода или вывода. Через эти порты вы передаёте и изменяете сигналы, чтобы получить  
необходимый результат. Логические компоненты работают на булевой основе — сигналы могут быть либо  
выключены (0), либо включены (1).

<br />

### Таблицы истинности

Логика будет вести себя по-разному в зависимости от входных сигналов, как правило, детерминированным образом.  
Поэтому у большинства логических компонентов есть так называемая таблица истинности. Таблица истинности показывает  
состояние выхода для каждой комбинации входов данного логического компонента. На странице каждого логического  
компонента, описываемого таблицей истинности, такая таблица будет присутствовать.

Входы обозначаются последовательно пронумерованными буквами. Например, первый вход — A, второй — B и так далее.  
Выходы обозначаются буквой Q. Если выходов несколько, после Q ставится индекс, например Q₁, Q₂ и так далее.  
Однако чаще всего у вентиля только один выход.

<br />

### Использование логики

Логические компоненты можно разместить или поднять в интерфейсе [Микрочипа](../microchips.md) левой кнопкой мыши.  
Чтобы настроить параметр компонента (если он есть), нажмите по нему правой кнопкой — откроется меню с настройками.  
Описания параметров каждого компонента находятся на его странице.

Чтобы узнать, как соединять компоненты проводами через их порты, смотрите страницу про [Провода](../redstone_bits.md).
