Редстоун-биты используются для создания проводов. Эти провода соединяют порты. 
Порты — это треугольные выступы на боковой стороне [Логических компонентов](logic/introduction.md). 
Выходные порты всегда справа, а входные — слева.

К каждому входному порту можно подключить только один провод. Количество проводов, выходящих из выходных портов, не ограничено. 
Если нужно объединить несколько проводов в один входной порт, используйте [Вентиль ИЛИ](logic/or_gate.md).

Редстоун-биты работают по схожему принципу, что и редстоуновая пыль - сигнал может иметь силу от 0 до 15.
Сила 0 означает, что провод выключен, а любое значение от 1 и выше считается включённым состоянием.

### Работа с проводами

Провод создаётся левым щелчком по выходному порту, затем левым щелчком по нужному входному порту. 
Неподключённый провод можно отменить правым щелчком. Уже размещённые провода можно подобрать, используя левый щелчок.

Каждый провод стоит один редстоун-бит. Бит расходуется только после подключения провода ко входному порту.
